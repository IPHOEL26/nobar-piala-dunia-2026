APLIKASI NOBAR PIALA DUNIA 2026 - IPHOEL EDUMATH

Cara menjalankan:
1. Ekstrak file ZIP.
2. Buka folder hasil ekstrak.
3. Klik dua kali index.html.
4. Aplikasi akan terbuka di browser.

Fitur:
- Dashboard keluarga.
- Daftar grup dan negara.
- Tombol Lolos, Gugur, Aktif, dan Favorit.
- Jadwal fase grup awal.
- Input skor pertandingan.
- Bagan gugur manual.
- Agenda nobar keluarga.
- Export dan import backup JSON.
- Bisa dicetak.

Catatan:
Data grup dan fase turnamen mengikuti data yang diberikan dalam percakapan.
Jika data resmi berubah, aplikasi bisa diperbarui kembali.
